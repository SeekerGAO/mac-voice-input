# README Screenshot Template

Use this section near the top of `README.md` after the short project summary.

中文版可放在 `README.zh-CN.md` 对应位置。

## Recommended Markdown

```md
## Screenshot

![MacVoiceInput Screenshot](./docs/images/screenshot-main.png)

_Suggested capture: menu bar icon, bottom floating HUD, waveform bars, and live transcript._
```

## Suggested Image File

- Path: `docs/images/screenshot-main.png`
- Format: PNG
- Width: 1600px to 2200px
- Keep the floating HUD clearly readable

## Suggested Capture Layout

- macOS desktop with the app visible in the menu bar
- Bottom-centered floating HUD in recording state
- Transcript text with Chinese or mixed Chinese-English content
- Optional second screenshot for the settings window
